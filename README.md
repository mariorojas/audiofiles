# Programmatically podcast creation 

**INSTALLATION COMMANDS**

```
git clone https://github.com/mariorojas/audiofiles.git
cd audiofiles/
python3 -m venv venv
. venv/bin/activate
pip install -r requirements.txt
```

To create an audio file, run the following:

```
python main.py
# output: podcast.mp3...
```
